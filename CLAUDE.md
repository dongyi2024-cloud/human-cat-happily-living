项目说明:
这是一个基于多智能体建模（ABM）的人宠共居住宅空间分析工具。通过 Python 仿真人与猫的时空轨迹，提取多维度评价指标（强度、熵、共现密度），自动识别空间冲突节点并辅助设计推导。

技术栈:
语言：Python 3.10+
数据处理：NumPy, Pandas
核心算法：Scikit-learn (DBSCAN 聚类), SciPy (信息熵计算)
图像处理：Matplotlib (绘图), Pillow (底图解析)
仿真环境：离散栅格坐标系 (200 x 200 Grid)

工作规范：
核心原则：所有评价指标计算必须追溯到原始轨迹数据（Tick 级别），禁止对已聚合的热力图进行二次运算，以避免“伪共现”误差。
提交规范：每完成一个算法模块（如格栅化、聚类、可视化）立即 commit，message 格式：feat/fix/research: 描述。
模块化要求：轨迹产生逻辑与指标分析逻辑需解耦，分析脚本通过读取轨迹 CSV 文件运行。
坐标统一：所有坐标映射必须通过统一的缩放因子（scale_x/y）转换，确保像素坐标、栅格索引与实际物理距离一致。
可视化要求：绘图脚本需支持无界面后端（matplotlib.use('Agg')），确保在不同开发环境下均能输出图片。
